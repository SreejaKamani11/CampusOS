# 📖 Overview

CampusOS is a full-stack Smart Campus Platform that digitizes essential campus services through a single web application.

Students can:

- Order stationery
- Order food from the canteen
- Upload print jobs
- Manage cart
- Place orders
- Track orders

Administrators can:

- Manage users
- Manage stationery products
- Manage canteen menu
- Manage categories
- Manage print jobs
- Manage orders
- View dashboard statistics

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/SreejaKamani11/CampusOS.git
```

---

## Backend

```bash
cd backend
npm install
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=3000

DATABASE_URL=your_database_url

JWT_SECRET=your_secret_key

CORS_ORIGIN=http://localhost:5173
```
